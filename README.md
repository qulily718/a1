# 股票监控分析系统

一个智能股票监控应用，能够实时分析股票数据并提供买入/卖出信号提示。

## 功能特性

- 📈 实时股票数据获取
- 📊 技术指标分析（MA、RSI、MACD、布林带等）
- 🔔 智能买入/卖出信号提示
- 📉 可视化图表展示
- ⏰ 定时监控功能

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

```bash
streamlit run app.py
```

然后在浏览器中打开显示的URL（通常是 http://localhost:8501）

## 使用说明

1. 在界面中输入股票代码（例如：AAPL、TSLA、000001.SS等）
2. 选择监控周期（日线、周线等）
3. 系统会自动分析并显示：
   - 当前价格和涨跌幅
   - 技术指标图表
   - 买入/卖出信号提示
   - 交易建议

## 股票代码格式

### 美股
- 直接输入代码即可，如：`AAPL`（苹果）、`TSLA`（特斯拉）

### A股（中国）
- **上海股票**：`000001.SS` 或 `600519.SS`（6开头的股票）
- **深圳股票**：`000001.SZ` 或 `002594.SZ`（0或2开头的股票）
- 也可以直接输入6位数字，系统会自动识别

**注意**：A股数据使用 `akshare` 库获取，如果遇到问题，请确保已正确安装：
```bash
pip install akshare
```

### 港股
- 需要加 `.HK` 后缀，如：`00700.HK`（腾讯）
